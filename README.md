## Components

- **Navbar**: A navigation bar component.
- **Banner**: A banner component displaying a slideshow of images.
- **Container1**: A container component with a Swiper component for displaying cards.
- **Container2**: A container component with some slides.
- **Aboutus**: A component displaying information about the company or organization.
- **Partners**: A section showcasing partner logos.
- **ContactUs**: A component providing contact information and buttons for booking a call or raising a query.
- **Trainer**: A trainer's page component with sub-components:
  - **AboutTrainer**: Displays information about the trainer.
  - **Whyjoinourclass**: Provides reasons to join the trainer's class.
  - **SessionTimeline**: Displays a timeline of training sessions.
  - **Team**: Displays the trainer's team members.
  - **Footer**: A footer component.

## Development

- The project uses Vite as the build tool.
- Styles for each component are defined in separate CSS files.
- React components are written in JSX and stored in the `src/components` directory.
- The main component that renders other components is `App.jsx`.
- Modify the components or add new components as per your requirements.
- Update the styling in the respective component's CSS file.
